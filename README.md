# project-koff-ma

## Setup
Add "individual_dev_settings.xml" to app/src/main/res/values/ with the following content:

```
<resources>
    <string name="rest_api_url">(IP address of your dev server)</string>
    <string name="google_api_key">(Your API Key)</string>
</resources>
```
